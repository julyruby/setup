setup.rb

This little ruby script automates the process of setting up the .pairs file as well as includes a couple neat atom addons. Let me know if there are anyother packages for atom you guys find useful!

  1. Click the 'Download' button below!
  2. Enter 'ruby setup.rb' in the terminal from the directory where you downloaded it to (likely '~/Downloads')
  3. Follow the prompts by entering your input and hitting the enter key.
